# 주식 대시보드

실시간 주식 시세, 섹터 분류, 거시경제 지표를 한눈에 볼 수 있는 개인용 주식 대시보드입니다.

## 기술 스택

- **백엔드**: FastAPI (Python) + yfinance + FRED API + OECD SDMX API
- **프론트엔드**: Vanilla JS + Chart.js
- **데이터베이스**: SQLite (모든 설정·순서·레이아웃 영구 저장)
- **데이터 소스**: Yahoo Finance, FRED, OECD

---

## 주요 기능

### 1. 주식 종목 관리
- 티커 또는 한국어 종목명으로 검색 (예: `AAPL`, `삼성전자`, `005930.KS`)
- 미국 주식 / 한국 주식(KOSPI · KOSDAQ) 모두 지원
- 종목 카드 삭제 및 드래그 앤 드롭 순서 변경 (마우스 · 터치 모두 지원)
- 추가한 종목과 레이아웃 순서는 SQLite DB에 저장되어 서버 재시작 후에도 유지

### 2. 종목 카드
각 종목마다 카드 형태로 다음 정보를 표시합니다.

| 항목 | 설명 |
|------|------|
| 종목명 · 티커 | 회사명과 티커 심볼 |
| 섹터 배지 | 해당 종목이 속한 SPDR 섹터 ETF (예: `XLK · 기술`) |
| 현재가 | 한국 주식은 ₩원화, 미국 주식은 ₩원화 환산가 + $달러가 함께 표시 |
| 등락폭 | 전일 정규장 종가 대비 변동액 및 변동률 |
| PER / Forward PER | 현재 PER과 예상 PER (한국 주식도 EPS 기반으로 계산) |
| Forward PER 기준 적정가격 | Forward PER 대비 현재 PER로 산출한 적정가격 및 괴리율 |
| Graham 적정가격 | `√(22.5 × EPS × BPS)` Graham 공식 기반 적정가격 |
| 기간별 수익률 | 5일 / 1달 / 3달 / 6달 / 1년 수익률 (장중 실시간 반영) |
| vs 섹터 ETF 초과수익 | 각 기간별로 해당 섹터 ETF 수익률 대비 초과/미달 수익률 |

카드 클릭 시 캔들스틱 차트 모달이 열립니다 (5일 / 1개월 / 3개월 / 1년 선택 가능).

### 3. 섹터별 그룹 분류
- 종목이 속한 섹터 ETF에 따라 자동으로 그룹화되어 표시
- 각 섹터 그룹 헤더에 5일~1년 ETF 수익률 표시
- 헤더 클릭으로 그룹 접기/펼치기

### 4. 상단 고정 바 (Sticky Header)

#### 주요 지수
- S&P 500, 나스닥, 다우존스, 러셀 2000, 필라델피아 반도체 지수 실시간 표시
- 드래그 앤 드롭(마우스) / 꾹 누르기 후 드래그(터치)로 순서 변경, DB에 저장
- 모바일에서 좌우 스와이프로 전체 지수 탐색 가능

#### 섹터 ETF 바
11개 SPDR 섹터 ETF를 당일 수익률 기준 내림차순으로 정렬 표시합니다.

| 티커 | 섹터 |
|------|------|
| XLK | 기술 |
| XLF | 금융 |
| XLV | 헬스케어 |
| XLY | 자유소비재 |
| XLP | 필수소비재 |
| XLI | 산업재 |
| XLC | 커뮤니케이션 |
| XLU | 유틸리티 |
| XLRE | 부동산 |
| XLB | 소재 |
| XLE | 에너지 |

수익률은 `regular_market_previous_close`(정규장 전일 종가) 기준으로 계산하여 정확한 당일 변동률을 표시합니다.

### 5. 섹터 ETF 수익률 추이 차트
- 11개 섹터 ETF를 하나의 차트에 정규화(기준=100)하여 비교
- 기간 선택: 1개월 / 3개월 / 6개월 / 1년 / 3년 / 5년
- 우측 범례에 각 ETF의 기간 수익률을 내림차순으로 표시
- 차트 헤더 클릭으로 섹션 접기/펼치기

### 6. 거시경제 지표 바
- 검색으로 원하는 지표를 추가하고 드래그 앤 드롭(마우스 · 터치)으로 순서를 바꿀 수 있습니다.
- **분류(카테고리)** 기능: 지표를 원하는 분류에 묶어 가로 열(column)로 표시
  - 지표 추가 시 기존 분류 선택 또는 새 분류 직접 입력
  - **분류 관리** 버튼 → 분류 레인 패널에서 지표별 분류 일괄 변경
  - 분류는 가로로 추가되고, 분류 내 지표는 세로로 쌓임
  - 제목 클릭으로 지표 영역 접기/펼치기
- 모든 설정은 DB에 저장되어 서버 재시작 후에도 유지

| 카테고리 | 지표 |
|----------|------|
| 금리 | 미국 국채 3개월 / 5년 / 10년 / 30년 |
| 통화 | 달러 인덱스 DXY |
| 환율 | 달러/원, 달러/엔, 달러/위안, 유로/달러 |
| 심리 | VIX 공포지수, MOVE 채권변동성 |
| 원자재 | 금, WTI 원유, 천연가스, 구리, 은 |
| 암호화폐 | 비트코인, 이더리움 |

### 7. 거시경제 분석 뷰 (기본 화면)
앱 시작 시 기본으로 표시되는 분석 화면입니다. 각 섹션을 드래그 앤 드롭(마우스 · 터치)으로 원하는 순서로 배치할 수 있습니다.

#### 장단기 금리 스프레드 (10Y - 2Y)
- 국채 수익률 곡선(2Y · 10Y · 30년)과 10Y-2Y 스프레드를 동시 표시
- 스프레드 구간을 4단계 배경색으로 시각화:

| 구간 | 의미 | 색상 |
|------|------|------|
| +0.5% 이상 | 정상 (경기 확장 신호) | 초록 |
| 0 ~ +0.5% | 평탄 (경기 둔화 우려) | 노랑 |
| -0.5 ~ 0% | 역전 (경기 침체 경고) | 주황 |
| -0.5% 미만 | 심한 역전 (경기 침체 위험) | 빨강 |

#### 섹터 로테이션 히트맵
- 11개 섹터 ETF의 1주 / 1개월 / 3개월 / 6개월 / 1년 수익률을 히트맵으로 표시
- 컬럼 헤더 클릭 시 해당 기간 기준 정렬

#### 공포 지수 (VIX)
- VIX 캔들스틱 차트 (최대 5년)
- 구간별 배경색 및 상태 배지 (안정 / 주의 / 공포 / 극도의 공포)

#### 필라델피아 연준 제조업지수
- Philadelphia Fed Manufacturing General Activity Index (FRED 데이터)
- 0 기준선 기준 확장/수축 구간 배경색 표시

#### 경기선행지수 OECD CLI
- OECD Composite Leading Indicator (CLI) — 미국 데이터
- 100 기준선 기준 확장/수축 구간 배경색 표시

#### 미국 TGA (재무부 일반계좌)
- FRED `WTREGEN` 데이터 기반 주간 잔액
- 잔액 수준에 따라 유동성 상태 배지 표시

### 8. 과거분석 뷰
보유 종목 전체를 한 차트에 정규화(기준=100)하여 기간별 수익률을 비교합니다.

- **기간 선택**: 1일 / 7일 / 1개월 / 3개월 / 6개월 / 1년 / 5년 / 10년 / 20년
- **날짜 직접 설정**: 시작일~종료일을 직접 입력하여 임의 기간 조회 (모바일 달력 지원)
- **높이 조절**: 소/중/대 버튼으로 차트 높이 즉시 변경, DB에 저장
- **범례 패널**: 기간 수익률 표시, 눈 아이콘으로 종목별 표시/숨김 전환
- **반응형**: 모바일에서 범례가 차트 아래로 이동하여 2열 표시
- **투자 수익 계산**: 투자금액 입력 시 보유 종목별 수익 시뮬레이션

### 9. 마인드맵 뷰
종목 간 관계를 자유롭게 시각화할 수 있는 캔버스 기반 마인드맵입니다.

- **종목 추가**: 검색창에서 종목명 또는 티커 검색 후 추가
- **카테고리 그룹**: 종목들을 카테고리 박스로 묶어 테마/섹터별 정리
- **연결선(Edge)**: 연결 모드에서 노드 클릭으로 종목·카테고리 간 관계 시각화, 라벨 편집 가능
- **수익률 표시**: 각 종목 노드에 1일 / 7일 / 1개월 / 6개월 / 1년 수익률 표시 (열람 시 실시간 갱신)
- **자유 배치**: 노드 드래그로 원하는 위치에 배치, 무한 캔버스(6000×6000) 위에서 팬·줌 가능
- **모드 전환**: 일반(이동) / 연결 / 삭제 모드 전환 버튼 제공
- **영구 저장**: SQLite DB에 자동 저장
- **터치 지원**: 핀치 줌·한 손가락 팬·롱프레스 등 모바일 제스처 지원

### 10. 자동 갱신
- 60초마다 모든 가격 자동 갱신 (상단 카운트다운 표시)
- 새로고침 버튼으로 즉시 갱신 가능

## 수익률 계산 방식

| 구분 | 기준가 | 현재가 |
|------|--------|--------|
| 당일 변동률 | `regular_market_previous_close` (정규장 전일 종가) | 실시간 현재가 |
| 기간별 수익률 | N일 전 종가 (조정가 기준) | 실시간 현재가 (장중 반영) |

- 모든 수익률은 장중에도 실시간 가격을 반영하여 계산합니다.
- 당일 변동률은 after-hours 가격이 아닌 정규장 전일 종가를 기준으로 합니다.

## 데이터 저장 구조

모든 사용자 설정과 레이아웃은 `localStorage` 없이 **SQLite DB에만** 저장됩니다.

| 저장 항목 | DB 키 |
|-----------|-------|
| 종목 목록 | `/api/db/stocks` |
| 지수 바 순서 | `indexBarOrder` |
| 매크로 지표 목록 | `macroIndicators` |
| 거시경제 섹션 순서 | `mavSectionOrder` |
| 마인드맵 데이터 | `mindmapData` |
| 과거분석 차트 높이 | `gvChartHeight` |

## 설치 및 실행

### 요구사항
- Python 3.9 이상

### 설치

```bash
git clone https://github.com/junyoo96/stock-dashboard.git
cd stock-dashboard
pip install -r requirements.txt
```

### 실행

```bash
python -m uvicorn main:app --reload
```

브라우저에서 `http://localhost:8000` 접속

> 첫 실행 시 `dashboard.db` SQLite 파일이 자동으로 생성됩니다.

## API 엔드포인트

| 메서드 | 경로 | 설명 |
|--------|------|------|
| GET | `/api/search?q=` | 종목 검색 |
| GET | `/api/price/{symbol}` | 현재가 조회 |
| GET | `/api/chart/{symbol}?period=` | 캔들스틱 차트 데이터 |
| GET | `/api/chart/{symbol}?start=&end=` | 날짜 범위 차트 데이터 |
| GET | `/api/valuation/{symbol}` | PER · 적정가격 데이터 |
| GET | `/api/performance/{symbol}` | 기간별 수익률 (실시간 가격 기준) |
| GET | `/api/sector-chart?period=` | 섹터 ETF 정규화 차트 데이터 |
| GET | `/api/fx/usdkrw` | USD/KRW 환율 |
| GET | `/api/macro/search?q=` | 거시경제 지표 검색 |
| GET | `/api/yield-curve-history` | 장단기 금리 데이터 (FRED) |
| GET | `/api/vix-history` | VIX 캔들스틱 데이터 |
| GET | `/api/ism-pmi-history` | 필라델피아 연준 제조업지수 (FRED) |
| GET | `/api/lei-history` | OECD CLI 경기선행지수 |
| GET | `/api/tga-history` | 미국 TGA 잔액 (FRED) |
| GET | `/api/db/stocks` | 저장된 종목 목록 조회 |
| POST | `/api/db/stocks` | 종목 목록 저장 |
| GET | `/api/db/settings/{key}` | 설정값 조회 |
| PUT | `/api/db/settings/{key}` | 설정값 저장 |

## 스크린샷

> **거시경제 분석 뷰** (기본 화면): 장단기 금리 스프레드 · 섹터 히트맵 · VIX · 제조업지수 · OECD CLI · TGA
